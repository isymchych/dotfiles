# CLI tools
* `yazi` - file manager
* `unarchiver` - archive preview for `yazi`
* `ripgrep` - search in files
* `fd` - find files
* `tokei` - count lines of code
* `just` - task runner
* `gitui` - git UI
* `delta` - show git diffs
* `fzf` - fuzzy matcher
* `sad` - search/replace tool
* `wget`
* `htop`
